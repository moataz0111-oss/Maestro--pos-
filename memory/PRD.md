# PRD - نظام إدارة المطاعم والكافيهات (Maestro EGP)

## الإصدار 23 - 18 يناير 2026

## ✅ ما تم إنجازه اليوم:

### 1. إصلاح مشكلة المبيعات ✅
- إحصائيات متعددة الفترات (يومي/أسبوعي/شهري/إجمالي)
- إصلاح عرض المبيعات حسب طريقة الدفع

### 2. نظام إدارة اليوم والترحيل ✅
- إغلاق تلقائي للورديات بعد 24 ساعة
- ربط إغلاق الصندوق بإغلاق الوردية
- نافذة إدارة اليوم

### 3. تقرير يومي عبر البريد ✅
- قالب HTML احترافي

### 4. طباعة إيصال خصم الموظف ✅

### 5. صفحة إدارة النظام (SystemAdmin) ✅
- فحص صحة النظام
- إحصائيات قاعدة البيانات
- نسخ احتياطي
- `/system-admin`

### 6. 🆕 تطبيق قائمة العملاء (Customer Menu) ✅
**رابط:** `/menu/{tenant_id}` أو `/menu/default`

**الميزات:**
- عرض قائمة الطعام بشكل جميل (Mobile-first)
- تصفح الفئات والبحث عن المنتجات
- سلة مشتريات مع حفظ محلي
- إرسال طلب للمطعم (يظهر في الطلبات المعلقة)
- تتبع حالة الطلب
- دعم الدفع النقدي (البطاقة قريباً)

**زر "قائمة العملاء":**
- في Dashboard يمكن للمستخدم الحصول على رابط القائمة ومشاركته

**APIs الجديدة:**
| Method | Endpoint | الوصف |
|--------|----------|--------|
| GET | `/api/customer/menu/{tenant_id}` | جلب قائمة الطعام |
| POST | `/api/customer/auth/register/{tenant_id}` | تسجيل عميل |
| POST | `/api/customer/auth/login/{tenant_id}` | تسجيل دخول عميل |
| POST | `/api/customer/order/{tenant_id}` | إنشاء طلب |
| GET | `/api/customer/orders/{tenant_id}` | طلبات العميل |
| GET | `/api/customer/order/{tenant_id}/{order_id}` | تتبع طلب |
| GET | `/api/customer/menu-link` | جلب رابط القائمة |
