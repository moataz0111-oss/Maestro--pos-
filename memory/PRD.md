# PRD - نظام إدارة المطاعم والكافيهات (Maestro EGP)

## الإصدار 32 - 19 يناير 2026

## ✅ ما تم إنجازه في هذه الجلسة:

### 1. إضافة قسم الوصفات في نافذة "إضافة صنف جديد" ✅
تم تعديل صفحة المخزون لتشمل:
- **قسم الوصفة (المكونات)** يظهر عند اختيار نوع "منتج نهائي"
- **قائمة منسدلة قابلة للتمرير** تعرض جميع المواد الخام المتاحة
- **حقل الكمية** لإدخال الكمية يدوياً لكل مادة
- **زر إضافة (+)** لإضافة المكون للوصفة
- **عرض المكونات المضافة** مع إمكانية الحذف
- **حساب تكلفة الوحدة تلقائياً** من مجموع تكاليف المكونات

### 2. إصلاح API المواد الخام ✅
- تم توحيد endpoint `/api/raw-materials` لاستخدام جدول `inventory` مع `item_type: raw`
- الآن يُرجع جميع المواد الخام (27 مادة) بشكل صحيح

---

## كيف يعمل النظام:

```
صفحة المخزون → إضافة صنف جديد
        ↓
    نوع: مادة خام | منتج نهائي
        ↓
   (إذا منتج نهائي)
        ↓
   قسم الوصفة (المكونات)
        ↓
   اختر مادة خام من القائمة → حدد الكمية → اضغط +
        ↓
   المكونات تُضاف للقائمة مع حساب التكلفة تلقائياً
        ↓
   حفظ المنتج النهائي مع وصفته
```
